# Survey Information System

A full-stack web application that collects survey data from users and stores it securely in a MongoDB database using a RESTful POST API.

---

## Tech Stack

### Frontend
- React.js
- JavaScript (ES6+)

### Backend
- Node.js
- Express.js

### DataBase
- MongoDB
- Mongoose

---

## Project Flow Overview

1. User fills survey form in React UI
2. React sends form data using HTTP POST request
3. Express API receives request
4. Data is validated
5. Data is stored in MongoDB
6. Success response is sent back to frontend
